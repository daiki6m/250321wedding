# Canva 連携ガイド

席次表のデータを Canva で編集するための手順です。

## 1. データの準備
プロジェクトのルートディレクトリにある `guests_for_canva.csv` を使用します。このファイルには、全てのゲスト情報（名前、テーブル、肩書き、メッセージなど）が含まれています。

## 2. Canva での「一括作成」手順
1. Canva で席次表やメッセージカードのデザインを開きます。
2. 左側のサイドバーから「アプリ」を選択し、「一括作成」を検索して開きます。
3. 「データをアップロード（CSV）」をクリックし、`guests_for_canva.csv` を選択します。
4. デザイン上のテキストボックスを右クリックし、「データの接続」を選択して、対応する項目（Name, Table など）を紐付けます。
5. 「続行」→「ページを生成」をクリックすると、全ゲスト分のデザインが自動生成されます。

## 3. デザインのヒント
ウェブサイトのデザインに合わせるための設定値です：

### カラーパレット
- **Table A (Rose)**: `#d65b75`
- **Table B (Amber)**: `#f2a842`
- **Table C (Blue)**: `#6987cf`
- **Table D (Lavender)**: `#cea1d1`
- **Table E (Sage)**: `#87bda2`
- **Table F (Teal)**: `#0097b2`
- **メインオレンジ**: `#F39800`

### フォント
- **手書き風**: `Zen Kurenaido` (Canva にない場合は似たフォント「さわらび明朝」や「Rounded Mplus」などを検討してください)
- **明朝体**: `Shippori Mincho` または `Zen Old Mincho`

## 4. 注意点
- メッセージが長いゲストの場合、Canva 上で文字サイズを個別に調整する必要があるかもしれません。
- `Participation` 項目で「本人」や「応援席」をフィルタリングして使用することをお勧めします。
